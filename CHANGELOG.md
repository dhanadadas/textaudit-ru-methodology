# 🔄 История версий (Changelog)

## [2.0.0] - 2026-03-16 (Context-Aware Diagnostics)
**Архитектурный сдвиг.**
- **Изменено:** Отказ от 6 изолированных оценок-цифр (вода, спам, и т.д. по 100-балльной шкале).
- **Добавлено:** Внедрен жанровый аудит. Модель принимает контекст (`genre`, `audience`) и возвращает массивы рисков (`critical`, `warning`, `success`), распределяя проблемы по степени их влияния на бизнес-задачу.
- **Улучшено:** UI-переработан: теперь отображается единый "Индекс здоровья" и три колонки с диагнозами.

## [1.8.0] - 2026-03-12 (Detector MVP Rollback)
- **Исправлено:** Откат экспериментальной "многопроходной" архитектуры детектора ИИ к однопроходной (Single Pass) из-за проблем с коллизиями позиций при подсветке текста.
- **Улучшено:** Унификация ключей категорий в `patterns.php`. Переход на литерные индексы (S, L, M, E, P, R -> U, V, R, G, S, C, D).

## [1.5.0] - 2026-02-20 (Semantic Antiplagiat Pipeline)
- **Добавлено:** Интеграция с XML-выдачей поисковиков. Теперь текст сначала пробивается по классическим базам, а затем ИИ анализирует "смысловой плагиат", сопоставляя идеи текста с текстами из ТОП-3 выдачи.
- **Добавлено:** Вывод найденных источников (`urls`) и оценка глубокого рерайта.

## [1.0.0] - 2025-11-10 (Initial Release)
- Запуск базовой матрицы качества. Определение словесного мусора, "воды" и канцелярита с помощью LLM-промптинга вместо словарей.
- Релиз веб-интерфейса.
